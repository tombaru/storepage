# Второй выпускной проект LoftSchool
## Для установки приложения выполните следующие команды


```
 $ git clone https://github.com/tombaru/storepage.git storepage
```

```
 $ cd storepage
```

```
 $ bower install
```

```
 $ gem install compass
```

```
 $ gem install sass-globbing
```

```
 $ npm install
```

```
 $ gulp
```
